The data used in this web app is from the CORGIS Dataset Project and can be found here. The data in this website is about Astronaut missons and information from 1959 to 2019, so you won't find anything before 1959 and after 2019. In the profile for the astronauts it shares the their personal information like their name, gender, birth year, nationality, if they were in the military, and the group that they worked with. Now for this data set everything is recorded yearly, so if you are trying to use this website to try to figure a certain month, day, or week, it won't be possible with this data set

The data also shares their misson information such as their lifetime misson count, misson duration, and eta duration.

Something that the data set doesn't tell is its measure of the time for the misson duration so I did some research on some of the misson it's self to see how they correlate and I saw that the data set measure time using the amount of hours that it took. For example, I used the Mercury Atlast misson to see how it lined up and on the data set it says that it took 5 hours but on the misson duration on the source it says 4 hours and 55 minutes and 23 seconds, so they just rounded up with some of the data.
